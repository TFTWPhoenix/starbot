# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Observatory Release  | Yes |
| BotKit Release   | No                |
| Classic Release   | No |
| Original Release   | No                |

## Observatory Release
Observatory release is supported for the releases on GitHub and also the latest commits.

Please note that branches other than `main` are not supported.

You will also not receive support if using a pull-request branch for non-testing purposes.

## Reporting a Vulnerability

Open an issue with the "Security Vulnerability" template.
