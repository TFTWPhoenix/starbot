# Starbot
A Minecraft Bot designed for Kaboom.pw

## Version
### Previous Versions:
**Original Release (OR)** The Original 1.0 release of Starbot. (In early 2020)

**Classic Release (CR)** Referred to as 2.0 through 16.0 at the time. Now referred to as CR-1.0 through CR-15.0

**BotKit Release (BKR)** After "version reset" referred to as CR-1.0 through CR-2.3
### Current Version
**Observatory Release (OBR)** The version of Starbot this repo contains. Version will be referred to as OBR-<semver version> beginning with 1.0

Documentation will be found in the docs/ directory.
